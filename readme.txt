== Description ==

A simple wordpress plugin to Hide your Adminbar on your site front end for logged in users.  

If you want to rework with the code, fine take a look at here before start usging this code, It will be helpful to work with your own project.<a href="http://kvcodes.com/" target="blank" > Kvcodes.com </a> 

**Features**

* Easy install
* Easy to show hide from the plugins directory.

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'WP List PlugIns'
1. Click 'install'

**Manual Installation**

1. Upload folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
